HyBERT Model Output

The HyBERT model produces several outputs that are used for automated ICD coding, model interpretation, and evaluation.

Output
Shape
Description
Logits
(B, C)
Raw prediction scores for each ICD code before sigmoid activation.
Probabilities
(B, C)
Sigmoid probabilities representing the likelihood of each ICD code.
Predicted ICD Codes
(B, K)
Final ICD codes selected using a threshold (e.g., 0.5) or Top-K prediction.
Code-wise Attention Weights
(B, C, L)
Attention scores indicating which clinical words contribute to each ICD prediction.
Context Vectors
(B, C, H)
Code-specific contextual representations generated by the attention mechanism.
Hyperbolic ICD Embeddings
(C, D)
Learned Poincaré ball embeddings preserving the hierarchical structure of ICD codes.
UMAP Visualization

Two-dimensional visualization of hyperbolic ICD embeddings showing hierarchical clustering.

Interpretation

The HyBERT model outputs a probability for every ICD code and assigns the most likely diagnoses based on a predefined threshold or Top-K selection. In addition to prediction, the model provides code-wise attention weights that identify the most influential words in the clinical note for each ICD code, improving interpretability. The learned hyperbolic ICD embeddings preserve the hierarchical relationships among ICD codes, while UMAP visualization demonstrates meaningful clustering of related diseases. These outputs enable HyBERT to achieve accurate, interpretable, and hierarchy-aware multi-label ICD coding suitable for clinical decision support and healthcare data management.
